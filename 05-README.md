# Gida AI 🏠🇳🇬

**Your Community's Digital Voice — in Your Language**

Gida AI is a free, mobile-first AI assistant built for everyday 
Nigerians who communicate in Hausa, Yoruba, Igbo, or Pidgin English. 
The goal is simple — make technology accessible to people who are 
not fluent in English or not exposed to modern digital tools.

## The Problem
Millions of Nigerians cannot benefit from AI tools because 
they are only available in English. Farmers, traders, students, 
and artisans are left behind simply because of language.

## The Solution
Gida AI speaks YOUR language. Ask questions by typing or 
speaking — and get answers in the same language you used.

## Features
- 🗣️ Voice input and text-to-speech in 4 Nigerian languages
- 🌍 Supports Hausa, Yoruba, Igbo, and Pidgin English
- 📱 Digital Literacy Help
- 💰 Financial Guide
- 🌾 Farmers Assistant
- 📚 Education Helper
- 💼 Jobs and Skills Guide
- 📵 Offline mode support
- 📲 SMS feedback system
- 🎛️ Voice speed control

## Built With
- Next.js 15
- Tailwind CSS
- Claude AI by Anthropic
- Web Speech API
